# 03. 실습 준비 체크리스트

강의 실습을 시작하기 전에 아래를 확인하세요.

## 공통

- [ ] **Kubernetes 클러스터**에 접근 가능 (`kubectl get nodes` 성공)
- [ ] **KUBECONFIG**가 올바른 클러스터를 가리킴
- [ ] **Git**으로 tz-chatbot 저장소 클론 완료
- [ ] (선택) **Helm** 설치됨 — bootstrap·RAG·Dify 설치 시 사용

## 로컬에서 gateway/admin 실행할 때

- [ ] **Python 3.11+** 설치 (chat-gateway, chat-admin 공통)
- [ ] **chat-gateway**: `.env` 또는 환경변수 (DIFY_BASE_URL, DIFY_API_KEY, CHAT_GATEWAY_JWT_SECRET 등)
- [ ] **chat-admin**: `.env` 또는 환경변수 (동일 + DB 등). 로컬은 SQLite 가능
- [ ] (선택) **가상환경** 생성·활성화 후 `pip install -r requirements.txt`

## K8s 배포 실습할 때

- [ ] **네임스페이스** 생성 가능 (또는 이미 devops, rag, dify 등 존재)
- [ ] **RAG Secret** 준비: MinIO 접근 키, Gemini/OpenAI API Key (rag/README.md 참고)
- [ ] **DB**: chat_admin용 PostgreSQL이 있으면, DB 생성 및 접속 정보 준비 (README Database setup)

체크리스트를 만족하면 다음 실습(05-1, 05-2, 05-3)으로 진행할 수 있습니다.
