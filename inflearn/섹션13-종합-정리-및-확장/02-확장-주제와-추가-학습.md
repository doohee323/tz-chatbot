# 02. 확장 주제와 추가 학습

## 확장 주제

- **새 토픽 추가**: MinIO 경로·Qdrant 컬렉션·Ingestion Job/CronJob·RAG Backend Deployment를 토픽별로 하나씩 추가. docs/rag-multi-topic.md 패턴 참고.
- **하이브리드 검색**: 벡터 검색에 BM25 등을 섞는 방식 (저장소·RAG Backend 구현 확인)
- **증분 Ingestion**: INCREMENTAL=true 로 변경분만 재색인. 대용량 문서가 늘어날 때 유리
- **모니터링·알림**: Prometheus·Grafana, Loki·로그 검색, 알림 룰 추가
- **다중 리전·클러스터**: 여러 K8s 클러스터에 동일 구성을 배포하거나, Dify·RAG만 별도 클러스터로 분리하는 설계

## 추가 학습 자료

- **Kubernetes**: 공식 문서, “바로 쓰는 CI/CD on K8S” 등 실습 강의
- **RAG·LangChain**: LangChain 문서, RAG 설계 베스트 프랙티스
- **Dify**: Dify 공식 문서, 워크플로우·도구 연동
- **PostgreSQL·MinIO·Qdrant**: 각 제품 문서로 백업·튜닝·고가용성
