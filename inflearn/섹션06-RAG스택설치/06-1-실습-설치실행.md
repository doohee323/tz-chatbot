# 06-1. 실습: 설치 실행

## 목표

- RAG 스택(install.sh)을 실행해 Qdrant, RAG Backend, Ingestion 리소스를 K8s에 설치합니다.

## 전제

- Kubernetes 클러스터 접근 가능
- Ingress NGINX, MinIO가 이미 설치되어 있음 (bootstrap 1~2단계 완료 권장)

## 단계

### 1. rag 디렉터리에서 install.sh 실행

```bash
cd tz-chatbot/rag
bash install.sh
```

- KUBECONFIG가 설정되어 있어야 함
- 출력 메시지를 보며 에러 없이 완료되는지 확인

### 2. Pod 기동 확인

```bash
kubectl get pods -n rag
```

- Qdrant, rag-backend, rag-backend-drillquiz 등이 Running이 될 때까지 대기 (이미지 pull·기동에 따라 수 분 걸릴 수 있음)

### 3. 다음 실습으로

- **06-2. 실습: Secret 및 검증**에서 토픽별 Secret을 생성하고, Backend 로그·검색으로 검증합니다.
- Secret을 만들기 전에는 Backend가 API 키 오류로 실패할 수 있으므로, 06-2를 반드시 진행하세요.
